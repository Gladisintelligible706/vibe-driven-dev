## 🚀 What this app does

vibe-driven-dev helps you turn a rough product idea into a clear set of steps for coding agents. It gives you a simple way to move from “I want to build this” to a plan that is easier to follow.

Use it when you want to:

- break a messy idea into smaller parts
- create a starter file set for a project
- choose a stack before you start coding
- prepare handoff notes for an AI coding tool
- keep agent work more organized and safer

It is built for people who want to use coding agents without starting from a blank page

## 🧩 How it fits into your workflow

vibe-driven-dev works best when you have an idea and need structure. A common flow looks like this:

1. Write your idea in plain English
2. Let the app shape it into a plan
3. Review the plan and adjust it
4. Generate starter files and notes
5. Pass the result to a coding agent
6. Build from a clearer base

This saves time when you know what you want, but you do not want to explain it from scratch each time
